👷‍♂️Temario de Html🌐

HTML, que significa Lenguaje de Marcado de Hipertexto, es el estándar que se utiliza para crear y diseñar páginas web. Funciona como el esqueleto de cualquier sitio web, permitiendo estructurar el contenido mediante el uso de etiquetas. Estas etiquetas definen elementos como encabezados, párrafos, enlaces, imágenes y listas, entre otros. HTML es fundamental para el desarrollo web, ya que proporciona la base sobre la cual se aplican estilos CSS y comportamientos JavaScript, permitiendo así la creación de sitios web ricos y dinámicos. A lo largo de los años, HTML ha evolucionado significativamente, con HTML5 siendo la última versión mayor, introduciendo nuevas etiquetas y atributos que ofrecen más funcionalidades y permiten una mejor semántica y compatibilidad con dispositivos móviles.

1. Presentando a HTML

🚨MUY IMPORANTE🚨, HTML es solamente un lenguaje de marcado (NO ES CONSIDERADO COMO UN LENGUAJE DE PROGRAMACIÓN ya que no nos permite realizar la declaración de variables, no tienes estructuras de control, etc.)
